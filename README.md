# Training Quiz App

Aplicación Flutter genérica, mínima y funcional para capacitación, evaluación y certificación.

## Configuración inicial

Debido a que este proyecto contiene únicamente el código fuente de Dart, debes ejecutar el siguiente comando en la raíz del proyecto para generar las carpetas nativas de Android, iOS, Web, etc.:

```bash
flutter create .
```

Luego, instala las dependencias:

```bash
flutter pub get
```

## Ejecución de pruebas

```bash
flutter test
```
